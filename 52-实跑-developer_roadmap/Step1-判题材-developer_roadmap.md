## 🪜 J-Ladder 架构分层体系

*   **L0 一句话本质**：
    开发者技术路线的本质是构建从物理硬件（内核/网络）到应用层软件（前端/后端）再到分布式云端（云原生/DevOps）的垂直拓扑知识网络，实现高可用、高扩展性系统的工程化交付。
*   **L1 四句话逻辑**：
    1. **底层操作系统支撑**：应用软件必须运行在操作系统之上，依托内核调度、虚拟内存和 I/O 复用机制实现与硬件的高效物理交互。
    2. **高并发后端协同**：基于关系与非关系存储、消息缓冲及分布式一致性共识，构建支撑大规模用户并发的服务端架构。
    3. **渲染前端与状态工程**：现代前端应用利用浏览器渲染流水线及 Virtual DOM Diff 机制，构建响应式交互和多端适配的用户层系统。
    4. **高可用云原生运维**：通过 Namespace/Cgroups 虚拟化和 Kubernetes 编排控制器，实现基础设施自动化、动态水平伸缩与系统自愈防御。
*   **L2 核心数据流转拓扑**：
    `用户浏览器交互` ➜ `V-DOM Diff 状态局部渲染` ➜ `CDN 缓存 / 边缘路由` ➜ `API 网关限流负载均衡` ➜ `Kubernetes Pod 副本池微服务` ➜ `Raft 共识协调` ➜ `分布式缓存（Redis） / 数据库（MySQL）`

## 🗂️ 28 张核心知识卡片大纲
1. 进程、线程与协程物理调度状态转换及上下文切换（Context Switch）开销。
2. 虚拟内存、物理内存页表与段页式 MMU 硬件地址翻译机制。
3. 进程间通信（IPC）机制：管道、匿名管道与共享内存信号量互斥锁。
4. 文件系统物理结构：inode 索引节点与虚拟文件系统（VFS）抽象模型。
5. Socket 网络协议栈套接字与五种系统级 I/O 模型（阻塞、非阻塞、多路复用、信号驱动、异步）。
6. 关系型数据库事务隔离级别（ACID）与 MVCC/两阶段锁（2PL）并发冲突控制。
7. NoSQL 内存缓存系统：Redis 单线程事件循环与 AOF/RDB 数据持久化逻辑。
8. 分布式一致性共识协议：Raft 领导者选举与 Paxos 状态机数据复制机制。
9. 消息队列机制：Kafka 高吞吐顺序追加写与稀疏索引查找逻辑。
10. 微服务服务治理与高并发跨进程通信：gRPC 二进制编码与 Protobuf 序列化。
11. 现代浏览器渲染管线：DOM/CSSOM 构建、Layout 布局、Paint 绘制与 Composite 合成。
12. 前端虚拟 DOM 状态控制：Reconciliation 协调过程与 Diff 算法核心复杂度分析。
13. 构建与工程化工具：Webpack 构建模块图依赖与 Vite 基于 ESM 的极速冷启动。
14. 单页应用（SPA）前端路由拓扑设计与 Redux/Vuex 全局状态机流转。
15. Web 安全防御规范：跨站脚本攻击（XSS）、跨站请求伪造（CSRF）与跨域资源共享（CORS）。
16. 容器物理隔离技术：Linux Namespace 命名空间与 Cgroups 硬件资源配额限制。
17. 容器编排架构：Kubernetes 核心控制器（Deployment/ReplicaSet）声明式 API 设计。
18. 持续集成与持续部署（CI/CD）自动化管线：Docker 构建缓存优化与发布策略。
19. 高可用健康检查探针设计：Kubernetes Liveness 存活探针与 Readiness 就绪探针。
20. 分布式一致性哈希（Consistent Hashing）算法及虚拟节点消除负载倾斜机制。
21. 高并发系统降级与熔断防护：Hystrix 舱壁隔离机制与熔断状态机转换。
22. 系统限流防护算法：漏桶（Leaky Bucket）算法与令牌桶（Token Bucket）算法。
23. 数据库水平拆分与 Shard Key 路由：分库分表与分布式全局唯一 ID（Snowflake）。
24. 系统可监控性（Observability）三位一体：指标（Metric）、日志（Log）与追踪（Trace）。
25. 领域驱动设计（DDD）战略与战术建模：限界上下文（Bounded Context）与聚合根（Aggregate Root）。
26. 测试驱动开发（TDD）规范：红-绿-重构（Red-Green-Refactor）环路与单元测试编写。
27. 无服务器架构（Serverless）冷启动优化：函数计算（FaaS）生命周期拦截管理。
28. 人工智能并网与 LLMOps 运维：检索增强生成（RAG）与向量数据库检索拓扑。
