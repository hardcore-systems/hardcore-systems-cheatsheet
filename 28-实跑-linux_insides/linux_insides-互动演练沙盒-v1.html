Linux 内存子系统物理分配与 SLUB 回收器仿真实验室

实时演示伙伴分配系统页框分裂与伙伴块合并，以及 SLUB 分配器 kmem_cache_cpu 局部无锁对象高速缓存的分配逻辑

实验一：伙伴页分配系统 (Buddy System Allocator)

支持分配 16KB 至 512KB 大小的物理内存块。伙伴页分配器会自动向下拆分高阶空闲页块（递归二分对半分裂）。释放时会自动检查相邻的 Buddy 块，若均空闲则合并为更高阶页块。

物理内存页地址映射大图 (1024KB)

0x0000 (0KB) 0x0100 (256KB) 0x0200 (512KB) 0x0300 (768KB) 0x0400 (1024KB)

实验二：SLUB 对象高速缓存分配 (SLUB Allocator)

模拟 SLUB 缓存管理对象。分配 64 字节对象（如进程描述符 task_struct）。快速路径读取 cpu-local 活跃高速缓存；当活跃 slab 耗尽时，触发慢速路径：向伙伴系统（实验一）申请一个 64KB 页框并完成 active 重建。

Node-Local Partial Slabs (kmem_cache_node->partial) 0 Slabs
