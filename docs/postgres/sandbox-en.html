Lab 2: SSI Serializable Anti-Dependency Cycle Detector

Serializable Snapshot Isolation (SSI) tracks read-write anti-dependencies (where transaction A reads data that transaction B subsequently writes). If a cycle of anti-dependencies is detected (e.g., T1 -> T2 -> T3 -> T1), a transaction must abort to prevent write skew.
